1.Difference between slice() and splice()
Feature	slice()	splice()
Purpose	Returns a portion of an array	Adds/Removes elements in the original array
Modifies Original Array?	❌ No (non-destructive)	✅ Yes (destructive)
Return Value	Returns a new array	Returns the removed elements
Syntax	array.slice(start, end)	array.splice(start, deleteCount, item1, item2, ...)
Example	js<br>let a = [1,2,3,4,5];<br>let b = a.slice(1, 4); // [2,3,4]<br>console.log(a); // [1,2,3,4,5]<br>	js<br>
let a = [1,2,3,4,5];<br>a.splice(1, 2); // removes 2 elements from index 1<br>console.log(a); // [1,4,5]<br>

In short:
•slice() → Extracts, no change to original.
•splice() → Cuts and can insert, modifies original.\

2.How do you convert an array to a string? And string to array?
Array to String
You can use:
•	toString() or
•	join() (you can specify a separator)
let arr = ['Apple', 'Banana', 'Mango'];
let str1 = arr.toString();    // "Apple,Banana,Mango"
let str2 = arr.join(' - ');   // "Apple - Banana - Mango"


String to Array
You can use:
•	split() method
let str = "Apple,Banana,Mango";
let arr = str.split(",");   // ['Apple', 'Banana', 'Mango']

4. What happens if you use `selectOption()` on a custom dropdown built with `<div>` instead of `<select>`?**

**Answer:**
`selectOption()` only works on native `<select>` tags.
 If you try to use it on custom dropdowns (e.g., `<div class="dropdown">`), it **won’t throw an error**, but it will **silently do nothing** or throw **"Element is not a <select>"** error.

await page.locator('#menu').hover(); // Hover to reveal dropdown
await page.locator('.dropdown-item', { hasText: 'Option 1' }).click();

**Explanation:**

6. How do you select multiple options from a `<select multiple>` dropdown?**

**Answer:**

await page.selectOption('#fruits', [
  { label: 'Apple' },
  { value: 'banana' },
  { index: 2 }
]);

**Explanation:**
Pass an array of options to `selectOption()`. This works only if `<select> multiple attribute` is present.

8.A dropdown option is hidden behind a scrollable container. How will you select it?**

await page.locator('.dropdown-container').scrollIntoViewIfNeeded();
await page.locator('.dropdown-item', { hasText: 'Hidden Option' }).click();

Use `scrollIntoViewIfNeeded()` to ensure the option is in view before clicking it.

EX:
import {test} from '@playwright/test'
test('clicking scrollable dropdown', async({page})=> {
await page.goto("https://codepen.io/m________________d/pen/gOOMeQN");

const farme=await page.frameLocator(`//iframe[@id="result"]`);
await farme.locator(`//button[@data-toggle="dropdown"]`).click();
await page.waitForTimeout(3000);
await farme.locator(`//a[text()='What not']`).scrollIntoViewIfNeeded();
await page.waitForTimeout(3000);
await farme.locator(`//a[text()='What not']`).click();
await page.waitForTimeout(2000)
})---

9. What’s the difference between using `.click()` on an option vs `selectOption()`? When should you choose one over the other?**

**Answer:**

* `.click()` is used for **custom dropdowns** (built with `<ul>/<div>`)
* `selectOption()` is for **native `<select>`** only

✅ Use `.click()` for modern styled dropdowns
✅ Use `selectOption()` for traditional forms

10. How can you write a dynamic function to select any option from a custom dropdown?**

**Answer:**

```ts
async function selectCustomDropdown(page, dropdownSelector, optionText) {
  await page.locator(dropdownSelector).click();
  await page.locator('.dropdown-item', { hasText: optionText }).click();
}

await selectCustomDropdown(page, '#createLeadForm_generalCountryGeoId', 'Canada');
```

**Explanation:**
Reusable method improves test readability and works for dynamic options.
